DontDelay URL Saver 개인정보처리방침

1. 수집하는 정보
- 사용자가 직접 입력한 DontDelay 연동 토큰(인증 정보)
- 사용자가 저장을 요청한 웹페이지의 URL, 페이지 제목, 선택 입력 메모
- 연결 설정용 포트 번호 및 마지막 저장 기록(기기 내 저장)

2. 수집 방법
- 사용자가 확장 팝업, 단축키, 컨텍스트 메뉴로 저장을 실행할 때만 수집합니다.

3. 이용 목적
- 로컬 PC에 설치된 DontDelay 데스크톱 앱의 URL 보관함에 링크를 저장하기 위해서만 사용합니다.

4. 저장 및 전송
- 토큰·설정은 Chrome 확장 프로그램의 로컬 저장소(chrome.storage.local)에만 저장됩니다.
- URL·제목·메모는 사용자 PC의 localhost(127.0.0.1)에서 실행 중인 DontDelay 앱으로만 전송됩니다.
- 개발자 서버, 제3자, 광고 네트워크로 데이터를 전송하지 않습니다.

5. 보관 기간
- 확장 설정 및 로컬 저장 데이터는 사용자가 삭제하거나 확장을 제거할 때까지 기기에 보관됩니다.
