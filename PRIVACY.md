# DontDelay 개인정보처리방침

**시행일:** 2026년 6월 4일  
**적용 대상:** DontDelay 데스크톱 앱(Windows · macOS), DontDelay URL Saver Chrome 확장 프로그램

---

## 1. 개요

DontDelay(이하 "본 서비스")는 학습·업무 계획을 돕는 데스크톱 애플리케이션입니다.  
본 서비스는 **사용자가 작성한 학습·업무 데이터를 사용자 기기 안에 보관하는 것**을 기본 원칙으로 합니다.

**할 일, 일정(캘린더), URL 보관함, 회고록**에 사용자가 입력·저장한 내용은 **사용자 PC의 로컬 파일에만 저장**되며, DontDelay 운영자나 제3자에게 **자동으로 전송·동기화·판매되지 않습니다.**  
따라서 해당 데이터가 인터넷을 통해 외부로 유출될 염려는 없습니다.

---

## 2. 수집·저장하는 정보

### 2.1 로컬에만 저장되는 정보 (외부 전송 없음)

다음 정보는 사용자가 직접 입력하거나 저장 동작을 실행할 때만 생성되며, **사용자 기기의 문서 폴더**에 JSON 파일로 저장됩니다.

| 구분 | 저장 내용 | 저장 위치 (예: Windows) |
|------|-----------|-------------------------|
| **할 일** | 제목, 메모, 상태, 마감일, 태그 등 | `%USERPROFILE%\Documents\DontDelay\todos.json` |
| **일정(캘린더)** | 할 일에 지정한 날짜·시간 정보 (별도 서버 없음, `todos.json` 기반) | 위와 동일 |
| **URL 보관함** | URL, 제목, 메모, 폴더, 출처 등 | `...\DontDelay\urls.json`, `url_folders.json` |
| **회고록** | 날짜, 제목, 본문, 이모지, 태그 | `...\DontDelay\retrospectives.json` |
| **태그** | 할 일·회고록용 태그 목록 | `...\DontDelay\tags.json` |
| **브라우저 연동 설정** | 로컬 API 포트, 연동 토큰 | `...\DontDelay\connection.json` |
| **앱 설정** | 테마(라이트/다크) 등 | OS·앱 로컬 설정 저장소 |

위 데이터는 **클라우드 백업, 원격 분석, 광고, 제3자 제공 목적으로 사용되지 않습니다.**

### 2.2 회원 기능을 이용할 때만 처리되는 정보 (선택)

회원가입·로그인을 **선택적으로** 이용하는 경우에 한해, 계정 확인을 위해 다음 정보가 **DontDelay 운영 서버**로 전송될 수 있습니다.

- 아이디(username), 비밀번호(서버에서 암호화·보관)
- 실명, 이메일, 학과, 전공 등 프로필 정보

로그인 세션 쿠키는 사용자 기기의 `Documents\dontdelay_cookies` 폴더에 로컬로 보관됩니다.

**중요:** 회원 정보는 계정·인증 목적에만 사용되며, **할 일·일정·URL·회고록 내용과는 분리**되어 있고, 위 2.1 항목의 데이터가 회원가입·로그인과 무관하게 외부로 올라가지 않습니다.

로그인 없이 앱을 사용하는 경우, 2.1 항목의 로컬 데이터만 처리됩니다.

### 2.3 Chrome 확장 프로그램 (DontDelay URL Saver)

확장 프로그램은 다음 정보만 처리합니다.

- 사용자가 입력한 **연동 토큰**, **포트 번호** → `chrome.storage.local`(기기 내)
- 사용자가 저장을 실행한 **웹페이지 URL·제목·선택 메모** → **localhost(`127.0.0.1`)에서 실행 중인 DontDelay 앱**으로만 전송

확장 프로그램은 **인터넷의 외부 서버로 사용자 데이터를 보내지 않습니다.**  
저장된 URL은 데스크톱 앱의 `urls.json`에 로컬로 기록됩니다.

## 3. 정보의 이용 목적

| 목적 | 해당 데이터 |
|------|-------------|
| 할 일·일정 관리 | 할 일, 캘린더(날짜) |
| 링크 보관·분류 | URL, 폴더, 메모 |
| 학습·활동 회고 작성 | 회고록 |
| 브라우저에서 URL 빠른 저장 | 확장 프로그램 → 로컬 앱 연동 |
| 계정 생성·로그인·프로필 표시 (선택) | 회원 정보 |

본 서비스는 위 목적 외로 개인정보를 이용하지 않으며, **맞춤 광고, 행동 추적, 데이터 판매**를 하지 않습니다.

---

## 4. 제3자 제공·국외 이전

- **할 일, 일정, URL, 회고록, 태그, 브라우저 연동 설정**은 제3자에게 제공되지 않습니다.
- 회원 기능 이용 시 전송되는 계정 정보도 **DontDelay 운영 목적 외 제3자 판매·제공을 하지 않습니다.**
- 본 서비스는 사용자 학습 데이터를 광고 네트워크·데이터 브로커 등에 전달하지 않습니다.

---

## 5. 보관 기간 및 파기

- **로컬 데이터:** 사용자가 앱에서 삭제하거나, `Documents\DontDelay` 폴더의 JSON 파일을 삭제하거나, 앱·확장 프로그램을 제거할 때까지 기기에 보관됩니다.
- **확장 프로그램 설정:** 확장 제거 또는 브라우저 데이터 삭제 시 함께 삭제됩니다.
- **회원 계정:** 탈퇴 요청 시 운영 정책에 따라 서버의 계정 정보를 삭제합니다. (로컬의 할 일·URL·회고록 파일은 사용자가 직접 관리합니다.)

## 6. 이용자의 권리

사용자는 언제든지 다음을 할 수 있습니다.

- 회원 탈퇴 및 계정 정보 삭제 요청 (회원 이용 시)
- 개인정보 처리에 관한 문의

## 7. 보안

- 학습·업무 콘텐츠는 **네트워크 전송 없이** 사용자 PC에만 저장됩니다.
- Chrome 확장 ↔ 데스크톱 앱 통신은 **로컬호스트(`127.0.0.1`)** 로만 이루어지며, 연동 토큰으로 무단 접근을 제한합니다.
- 회원 비밀번호는 평문으로 저장하지 않으며, 서버 측 보안 정책에 따라 처리됩니다.
